formula for Pascal triangle 

nth row has n columns -> index will be from 1 to n-1 (0 idx has 1)
temp = 1. 
for(int j=1;j<n;j++) {
    temp = temp * (n-j); 
    temp = temp/j; 
    ans.add(temp); 
}

Time complexity : O(n); 
Space complexity : O(1);
